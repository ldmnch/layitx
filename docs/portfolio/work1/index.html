COVID Social Representations

Is it possible to construct relevant information about social representations regarding a specific theme using news reader’s comments?

Which are the main topics discussed in news comments? How do these topics evolve over time? The aim of this paper is to present some preliminary results of an ongoing research addressing these questions using some novel data sources.

We presented a first approximation to the quantification of social representations about COVID-19, using news comments. A web crawler was developed to construct the dataset of reader’s comments. We detected relevant topics in the dataset using Latent Dirichlet Allocation, and analyze its evolution during time. Finally, we show a first prototype to the prediction of the majority topics, using FastText.
